## 背景

OpenCode（オープンソースのAIコーディングエージェント）を使って日々コーディングしている中で、こんなニーズが生まれました。

**「家のPCでAIにコードを書かせているが、外出先のスマホから結果を見たい・プロンプトを送りたい」**

OpenCodeはOpenRouterなどのLLMプロバイダーと連携してコード生成を実行しますが、その操作は基本的にPCのターミナル（TUI）上で行う必要があります。数分の料理の待ち時間や電車の中でも、AIにコードを書かせておいて結果だけ確認できれば——そんなモバイル連携の需要に応えるのが、今回紹介する構成です。

## 発生した問題

PCの前でしかOpenCodeを使えないという制約により、以下のような不便がありました。

- **PCから離れるとプロンプトを送れない**：AIにコードを書かせている最中にPCの前を離れると、次の指示が出せない
- **結果確認ができない**：生成が終わってもスマホで結果を見る手段がない
- **安全なリモートアクセスが必要**：単にポートを開放して外部公開するのはセキュリティ的に不安

## OpenCodeのリモートアクセス機能を比較する

OpenCodeには、リモートアクセスに関する複数の機能が用意されています。それぞれの特徴を比較してみましょう。

| 機能 | 説明 | プロンプト送信 | 結果確認 |
|------|------|:--------------:|:--------:|
| `opencode web` | ブラウザベースのフルTUI | ✅ | ✅ |
| `opencode serve` | HTTP API（REST + SSE） | ✅ (POST) | ✅ (GET/Stream) |
| Share link (`/share`) | 共有URLで会話を閲覧 | ❌ | ✅ |
| `opencode attach` | リモートサーバーにTUI接続 | ✅ | ✅ |

この中で、「スマホからプロンプトを送信したい」かつ「結果を確認したい」という両方の要件を満たす最適解は **`opencode web`** です。

## 解決策：OpenCode Web + Tailscale

### なぜこの組み合わせか

- **`opencode web`**：ブラウザ上でOpenCodeのTUIを完全に再現。スマホのブラウザからプロンプト送信・結果確認・セッション管理がすべて行えます
- **Tailscale**：WireGuardベースのメッシュVPNサービス。無料で使え、セットアップが非常に簡単。エンドツーエンド暗号化で安全な通信経路を確保します

以下、実際のセットアップ手順をステップバイステップで解説します。

## セットアップ手順

### ステップ1：Tailscaleのインストール

まずはPCとスマホの両方にTailscaleをインストールし、同一のアカウントで接続します。

**PC（Windows）の場合**：
- [Tailscale公式サイト](https://tailscale.com/download) からWindows版をダウンロードしてインストール
- インストール後、タスクトレイのTailscaleアイコンからアカウントにログイン

**スマホ（iPhone / Android）の場合**：
- App Store または Google Play で「Tailscale」を検索してインストール
- 同じアカウントでログイン

接続が完了すると、両方のデバイスに `100.x.x.x` 形式のTailscale IPが割り当てられます。PC側のTailscale IPは後でスマホからアクセスする際に使うので控えておきましょう。

```powershell
# PowerShellでPCのTailscale IPを確認
ipconfig | findstr "100."
```

### ステップ2：OpenCode Webサーバーを起動

PCのターミナルで、以下のコマンドを実行します。**パスワードの設定は必須**です。

```powershell
# 強力なパスワードを設定して起動
$env:OPENCODE_SERVER_PASSWORD = "your-strong-password"
opencode web --hostname 0.0.0.0 --port 4096
```

オプションの意味：
- `--hostname 0.0.0.0`：外部からの接続を許可（デフォルトはlocalhostのみ）
- `--port 4096`：任意のポート番号（空いていれば何番でも可）

パスワード未設定の場合は認証なしで誰でもアクセスできてしまうため、**必ず** `OPENCODE_SERVER_PASSWORD` 環境変数を設定してください。

### ステップ3：スマホからアクセス

スマホのブラウザで、以下のURLにアクセスします。

```
http://100.x.x.x:4096
```

（`100.x.x.x` は先ほど確認したPCのTailscale IPに置き換えてください）

Basic認証のダイアログが表示されるので、以下の情報を入力します。

| 項目 | 値 |
|------|-----|
| ユーザー名 | `opencode` |
| パスワード | 先ほど `OPENCODE_SERVER_PASSWORD` に設定した値 |

認証に成功すると、スマホのブラウザ上でOpenCodeのWeb UIが表示されます。

### ステップ4：実際に使ってみる

Web UIでは以下の操作が可能です。

- **セッション一覧の確認**：PCで実行中のセッションを一覧表示
- **新規プロンプトの送信**：スマホからAIへの指示を送信
- **結果の閲覧**：生成されたコードや応答を確認
- **既存セッションの継続**：過去の会話の続きからプロンプトを送信

家に戻ってPCで作業を再開する場合も、シームレスに移行できます。

```bash
# PCのTUIから同じサーバーにアタッチして継続
opencode attach http://localhost:4096
```

## 代替案：Cloudflare Tunnel

Tailscaleが使えない環境や、スマホにアプリをインストールできないケースでは、Cloudflare Tunnelを利用する方法もあります。

```bash
# cloudflared をインストールしてトンネル作成
cloudflared tunnel --url http://localhost:4096
```

さらに **Cloudflare Access** と組み合わせることで、Tailscale相当の認証レイヤーを追加できます。

ただし、Tailscaleに比べて以下の点で劣ります：
- セットアップがやや複雑
- 常時インターネット接続が必要
- Cloudflare Accessは無料枠の制限あり（50ユーザーまで無料）

**自宅PCと自分のスマホだけの接続であれば、Tailscaleの方が圧倒的にシンプルでおすすめです。**

## セキュリティ考慮事項

### パスワードは必ず設定する

`OPENCODE_SERVER_PASSWORD` を設定しない状態で `--hostname 0.0.0.0` を指定すると、**誰でも無認証でアクセスできる状態**になります。Tailscaleで経路は保護されていても、同じTailscaleネットワーク内の他のデバイスからアクセス可能になるため、必ず強力なパスワードを設定しましょう。

### Tailscaleのセキュリティ

- **エンドツーエンド暗号化**：TailscaleはWireGuardプロトコルを使用しており、すべての通信が暗号化されます
- **メッシュVPN**：中央サーバーを経由しないP2P接続のため、パフォーマンスが良い
- **無料プラン**：個人利用であれば6ユーザーまで無料で利用可能（2026年時点）

### 共有リンクの取り扱い注意

`/share` 機能で作成した共有リンクには、会話の内容が含まれます。企業の機密情報やAPIキーなどの認証情報が会話に含まれている場合は、共有リンクを作成しないように注意してください。必要に応じて、OpenCodeの設定でshare機能自体を無効にすることも可能です。

## 学び・補足

- **`opencode web` が最も簡単なリモートアクセス方法**：専用クライアント不要でブラウザだけで完結するため、スマホとの相性が非常に良いです
- **Tailscaleはリモートアクセスのベストプラクティス**：ポート開放不要、NAT越え対応、暗号化通信を無料で実現できます
- **外出先 → スマホ、帰宅後 → TUIでシームレス継続**：`opencode attach` を使えば、スマホで始めた作業をPCのTUIで続けることができます
- **`opencode serve` との使い分け**：カスタムBot（Telegram Botなど）からプログラム経由で操作したい場合は `opencode serve` のHTTP APIが適しています。人間がブラウザで操作するなら `opencode web` の方が直感的です

この構成を一度セットアップすれば、「PCの前でしかAIコーディングエージェントを使えない」という制約から解放され、場所を選ばない柔軟な開発フローを手に入れられます。
