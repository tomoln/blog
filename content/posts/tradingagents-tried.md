## TradingAgentsってなに

TradingAgentsは、TauricResearchが公開しているマルチエージェントLLM金融トレーディングフレームワークです。一言で言うと「複数のAIエージェントが協調して株式分析をしてくれるツール」です。

具体的には以下のエージェントたちが役割分担して動作します：

- **ファンダメンタル分析エージェント** — 財務諸表や決算データを読み解く
- **センチメント分析エージェント** — 市場のムードやSNSの反応を分析
- **ニュース分析エージェント** — 直近のニュースが株価に与える影響を評価
- **テクニカル分析エージェント** — チャートパターンやインジケーターをチェック
- **リサーチャー（強気派・弱気派）** — 分析結果を批判的に検証し議論
- **トレーダー** — 分析を総合して売買判断
- **リスク管理チーム** — ポジションサイズやリスク評価
- **ポートフォリオマネージャー** — 最終承認

これらが協調して最終的に **Buy / Overweight / Hold / Underweight / Sell** の5段階の判定を下します。

プロジェクトの詳細は [GitHub](https://github.com/TauricResearch/TradingAgents)（86K Stars、Apache-2.0ライセンス）で公開されており、[arXivの論文](https://arxiv.org/abs/2412.20138)も併せて公開されています。

## TradingAgentsって安全なのか

オープンソースの金融ツールということでセキュリティ面を実際に調査してみました。

### 脆弱性の履歴

過去にパストラバーサル（ティッカー名をファイルパスに使う際のバリデーション不足）の脆弱性が報告されましたが（PR #618）、既にv0.2.5で修正済みです。GitHubのSecurityタブを見ても公開されているSecurity Advisoryは現時点で0件。86K Starsの大規模プロジェクトであり、何か問題が見つかればIssuesやPRで報告・修正されるサイクルが機能しています。

### 今回の利用範囲ならリスクはほぼゼロ

TradingAgentsは株価分析を行い結果を表示するツールで、APIキーを環境変数に設定する程度の操作です。ペーパートレーディング（シミュレーション）に使う分には実質的にリスクはありません。もちろん、リアルマネーと連携するような改造をする場合は、APIキー管理や取引所の認証情報の取り扱いに注意が必要です。

## TradingAgents使ってみた（AAPLを分析）

実際にセットアップしてApple（AAPL）を本日日付で分析してみました。LLMには **deepseek-v4-flash** を **OpenRouter** 経由で使用しています。

### 分析結果：Hold（保有）

結論から言うと判定は **Hold** でした。強気でも弱気でもなく「現状維持」という判断です。

### テクニカル視点の判断根拠

実際の分析レポートから抜粋：

- **50日移動平均線（$285.36）と200日移動平均線（$266.46）**: 株価（$291.13）はいずれも上回っており、中長期的な上昇トレンドは維持
- **MACDヒストグラム（-3.51）**: 弱気クロス発生中で短期的な下落モメンタム
- **RSI（44.09）**: 中立よりやや弱い
- **ボリンジャーバンド**: 下限（$288.54）に接近しておりサポートが意識される水準
- **ATR（7.32）**: 高ボラティリティ状態

長期目線では買い材料があるものの、短期的なモメンタムが弱いため「今すぐ買うタイミングではない」という判断です。

### ファンダメンタル視点

- 年間1,000億ドル超のフリーキャッシュフロー（FCF）
- 大規模な自社株買いを継続中（EPS成長率+22.7%に貢献）
- サービス収益（App Store, Apple Music, iCloud等）の比率が拡大、粗利率49.3%への改善

### ニュース・センチメント

Reutersが報じたインドTata工場の環境汚染問題や、SpaceX IPOによる市場注目の分散などが短期的な懸念材料として挙げられました。

## API必要になってOpenRouterを試した

TradingAgentsを動かすにはLLMのAPIキーが必要です。私は普段 **opencode Go**（開発アシスタント）を使っていて、その内部では同じdeepseek-v4-flashが使われているのですが、TradingAgentsから直接それを呼び出すことはできません。

そこで調べたところ **OpenRouter** が便利だと分かりました。

### OpenRouterの良いところ

- **プリペイド式**: クレジットをチャージして使うだけ。使い過ぎの心配がない
- **リアルタイムの使用量ダッシュボード**: どのモデルにいくら使ったかが一目で分かる
- **豊富なモデル**: OpenAI、Anthropic、Google、DeepSeek、Mistralなど主要LLMが揃っている

そして何より驚いたのが **deepseek-v4-flashの価格** です。

| 項目 | OpenRouter価格 | DeepSeek公式価格 |
|------|---------------|-----------------|
| 入力 | $0.09 / 1Mトークン | $0.14 / 1Mトークン |
| 出力 | $0.18 / 1Mトークン | $0.28 / 1Mトークン |

**OpenRouterの方がDeepSeek公式より安い** というのも面白いポイントです。

## deepseek v4 flashはコスパ最強

今回の検証で改めて感じたのは、**deepseek-v4-flashのコストパフォーマンスの高さ** です。

マルチエージェント分析1回あたりのトークン消費量はおおよそ **入力500K＋出力15Kトークン** 程度（推定）。これをdeepseek-v4-flashのOpenRouter価格で計算すると：

```
入力: 500/1000 × $0.09 = $0.045
出力: 15/1000 × $0.18 = $0.0027
合計: 約$0.05（日本円で7〜10円）
```

1回の分析が約10円。実際に5ドル（約750円）チャージしておけば、100回以上の分析ができる計算になります。

性能面でも分析結果に破綻がなく、判断根拠も論理的に説明されていました。opencode Goのバックエンドとしても使われているモデルですが、API経由で直接使っても同等以上のパフォーマンスが得られます。個人でAIトレード分析を楽しむにはうってつけの選択肢です。

## まとめ

今回TradingAgentsを実際に動かしてみて得られた知見です：

1. **TradingAgentsはペーパートレーディングに特化した良ツール**: 複数エージェントが協調して分析するアーキテクチャは勉強になります
2. **deepseek-v4-flash + OpenRouter の組み合わせで格安運用**: 1回の分析が約10円。趣味の範囲で全く現実的
3. **投資判断のセカンドオピニオンとして使える（ただし鵜呑みにはしない）**: AIの分析はあくまで参考値。最終判断は自分で行いましょう
4. **お金をかけずに始められる**: TradingAgentsは無料のOSS、OpenRouterは最低5ドルから。気軽に試せます

AIエージェントが株を分析する——まだ発展途上の分野ですが、TradingAgentsはその最先端を手軽に体験できるプロジェクトです。
