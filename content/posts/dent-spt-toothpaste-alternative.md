## この記事でわかること

- DENT. SP-T（システマ）が販売終了になった経緯と後継品の値段
- SP-T の特徴をおさらい（乳酸菌LS1株・IPMP・フッ素・ジェルタイプ）
- 代替品として見つけた **コンクール ジェルコートF** のスペックと価格
- SP-T とコンクールの違い／選び分けのポイント

## 背景

数年前、歯医者で「これ使ってみて」と勧められたのが **DENT. SP-T（システマ）** でした。

歯科専売品のジェルタイプ歯磨き粉で、「口腔フローラを整える」という独自のコンセプトに惹かれて使い始めました。ジェルなので歯磨き粉の粉っぽさがなく、発泡剤も入っていないのでじっくり磨ける。研磨剤無配合なので歯にも優しい。

以来ずっとリピートしていて、Amazonで1本1,900円前後で買っていました。「歯磨き粉に2,000円は高いけど、歯科専売品だし仕方ない」と思いながらも、効果に満足していたので続けていました。

## 発生した問題

先日、いつものようにAmazonでSP-Tを注文しようとしたら、**商品ページが消えている**ことに気づきました。

検索しても出てくるのは見慣れないパッケージの商品。

![DENT. SP-T の商品ページ]()

…正確には、商品そのものがなくなっていたわけではなく、**DENT. SP-T が販売終了になり、後継品の「Systema SP-T ジェル Plus」に切り替わっていた**のです。

で、その値段を見て絶句。

**3,980円**。

今までの倍です。

## 原因（というか値段の話）

なぜ後継品が倍の値段になったのかはメーカーに聞かないとわかりませんが、想像するに：

- リニューアルに伴う開発コスト
- フッ素濃度が950ppm（パッケージ記載）から1,450ppmにアップグレード
- パッケージリニューアル

といった要素が価格に反映されたのでしょう。

ただ、いくらなんでも **「歯磨き粉に4,000円」は出せない**。

1本2〜3ヶ月持つとはいえ、年に2〜3本買うとして年間1万円超。さすがに躊躇するレベルです。

## 解決策：代替品を探す

「じゃあ何か代わりになるものはないのか」と調べ始めました。

まずは **DENT. SP-T の特徴**をおさらいしておきます。

| 項目 | DENT. SP-T | Systema SP-T ジェル Plus（後継） |
|------|-----------|-------------------------------|
| 価格 | 約1,900円 | **3,980円** |
| フッ素 | 950ppm | **1,450ppm** |
| 殺菌成分 | IPMP | IPMP |
| 乳酸菌 | SP-T乳酸菌（LS1株） | SP-T乳酸菌（LS1株） |
| タイプ | ジェル | ジェル |
| 研磨剤 | 無配合 | 無配合 |
| 発泡剤 | 無配合 | 無配合 |
| 販路 | 歯科専売品 | 歯科専売品 |

SP-Tの特徴は大きく3つ：

1. **SP-T乳酸菌（LS1株）** — 口腔フローラを整える独自技術
2. **IPMP** — 殺菌成分
3. **フッ素 + ジェルタイプ** — 虫歯予防と低刺激

「乳酸菌LS1株がどうしても欲しい」というなら SP-T Plus を買うしかありません。しかし、殺菌＋フッ素＋ジェルタイプ のベース機能だけでよければ、他の製品でも代用できるはず。

そう考えて探した結果、たどり着いたのが **コンクール ジェルコートF（ウエルテック）** です。

### コンクール ジェルコートF のスペック

| 項目 | コンクール ジェルコートF |
|------|----------------------|
| 価格 | **約1,000〜1,200円** |
| フッ素 | **1,450ppm**（SP-T Plusと同じ） |
| 殺菌成分 | **IPMP配合** |
| タイプ | ジェル |
| 研磨剤 | 無配合 |
| 発泡剤 | 無配合 |
| 販路 | 歯科専売品（ Amazon / 薬局で購入可） |
| レビュー評価 | **4.7〜4.9**（Amazon） |

SP-T Plus と比較してみましょう。

| 比較項目 | SP-T Plus（3,980円） | コンクール（1,000〜1,200円） |
|---------|-------------------|--------------------------|
| フッ素 | 1,450ppm | **1,450ppm（同じ）** |
| IPMP殺菌 | ✅ | ✅ |
| ジェルタイプ | ✅ | ✅ |
| 研磨剤無配合 | ✅ | ✅ |
| 発泡剤無配合 | ✅ | ✅ |
| 乳酸菌LS1株 | ✅ | ❌ |
| **価格** | **3,980円** | **約1,000円** |

**フッ素濃度、殺菌成分、ジェルタイプ、研磨剤・発泡剤無配合**といった基本スペックは完全に一致しています。

唯一の違いは **SP-T乳酸菌（LS1株）の有無**。これに価値を見出すかどうかが SP-T Plus を選ぶ基準になります。

ただし、この違いに年間1万円近くの差額を出すかと言われると、私は「出さない」という判断をしました。

## 学び・補足

### 歯科専売品の値段は「独自成分プレミアム」が乗っている

SP-T Plus が高い理由は、**独自の乳酸菌技術（LS1株）** に対するプレミアム価格です。この技術はライオン（システマ）の特許・独自研究によるもので、他社では真似できません。

一方、**「フッ素 + IPMP殺菌 + ジェルタイプ」** というベース機能だけ見れば、コンクールは同じスペックで1/4の価格を実現しています。

### コンクールの評判

Amazonのレビューを見ると、コンクール ジェルコートFは評価4.7〜4.9と非常に高い。

ユーザーの声としては：
- 「歯茎の状態が良くなった」
- 「ジェルタイプで磨きやすい」
- 「SP-Tの代替として使っている人が結構いる」

どうやら SP-T ユーザーが同じように代替品を探してコンクールにたどり着くパターンは珍しくないようです。

### 乳酸菌LS1株にこだわるなら

どうしても口腔フローラを整える乳酸菌の効果が欲しい場合、**SP-T Plus しか選択肢はありません**。

といっても、普通の歯磨き粉＋プロバイオティクス（口腔用乳酸菌タブレットなど）で似たような効果を狙う方法もあります。ただし、そこまでするなら SP-T Plus を買ったほうがトータルコストは安いかもしれません。

## 結論

私は **コンクール ジェルコートF** を試してみることにしました。

「SP-Tの乳酸菌LS1株、そんなに効いてた実感あったっけ？」と自問してみて、正直「よくわからなかった」というのが本音です。IPMPの殺菌効果とフッ素による虫歯予防、あとジェルタイプの磨きやすさに価値を感じていたので、それが同じスペックで約1/4の価格で手に入るなら乗り換える理由としては十分です。

**「SP-T Plus が高すぎて買えない」という同じ悩みを持つ方の参考になれば幸いです。**
